HTML (HyperText Markup Language)

HTML Elements

  • Elements ➔ A typical element includes an opening tag with some attributes, enclosed text content and a closing tag.
  • <Tags> ➔ A tag is used for creating an element. The name of an HTML element is the name used in angle brackets such as <p> for paragraph. The end tag's name is preceded by a slash character, </p>.
  • Attributes ➔ An attribute extends an element, changing its behavior or providing metadata. An attribute always has the form name="value" (the attribute's identifier followed by its associated value).
<Tag> / Attribute Name Description
<!DOCTYPE html>
<html> </html> 		Document Type Declaration
HTML Element		 Required as the 1st line of an HTML document (HTML 5 version).

The root of an HTML document.
<head> Head element It contains general information about an HTML page that isn’t displayed on the page itself. This information is called metadata and includes things like the title of the HTML document and links to stylesheets.
<title> Title Element It contains a text that defines the title of an HTML document. The title is displayed in the browser’s title bar or tab in which the HTML page is displayed. It can only be contained inside a document’s <head> element.
<body> Body Element It represents the content of an HTML document. Content inside are rendered on the web browsers.
<h1>-<h6> Heading Elements They are ordered from the highest level <h1> to the lowest level <h6>.
<p> Paragraph Element It contains and displays a block of text.
<span> Span Element It is an inline container for text and can be used to group text for styling purposes. However, as <span> is a generic container to separate pieces of text from a larger body of text, its use should be avoided if a more semantic element is available.
<div> Div Element It is used as a container that divides an HTML document into sections and is short for “division”. <div> elements can contain flow content such as headings, paragraphs, links, images, etc.
<br> Line Break Element It will create a line break in text and is especially useful where a division of text is required, like in a postal address. The line break element requires only an opening tag and must not have a closing tag.
<ol> Ordered List Element It creates a list of items in sequential order. Each list item appears numbered by default.
<ul> Unordered List Element It is used to create a list of items in no particular order. Each individual list item will have a bullet point by default.
<li> List Item Element It creates list items inside Ordered Lists or Unordered Lists.
<img> Image Element It embeds an image in the document. It is an empty element meaning it should not have a closing tag.
src Source Attribute It contains the image / video URL and is mandatory.
alt Alt Attribute The alternative text will be displayed if an image fails to render.
<video> Video Element It embeds a media player for video playback. The src attribute will contain the URL to the video.
controls Controls Attribute It will display video controls in the media player.
<a> Anchor Element It is used to create hyperlinks in an HTML document. The hyperlinks can point to other webpages, files on the same server, a location on the same page, or any other URL via the hyperlink reference attribute, href.
href Hyperlink Reference Attribute It determines the location the anchor element points to.
URL paths in HTML can be absolute paths , like a full URL , for example: 
https://developer.mozilla.org/en-US/docs/Learn
or a relative file path that links to a local file in the same folder or on the same server, for example: 
./style.css
.
target Target Attribute It specifies where a hyperlink should be opened. A target value of "_blank" will tell the browser to open the hyperlink in a new tab.
id Unique ID Attribute Specific and unique id attributes can be assigned to different elements in order to differentiate between them. Valid id attributes should begin with a letter and should only contain letters (a-Z), digits (0-9), hyphens (-), underscores (_), and periods (.).
The anchor element <a> can create hyperlinks to different parts of the same HTML document using the href attribute to point to the desired location with # followed by the id of the element to link to.

Semantic HTML

<Tag> / Attribute Name Description
<header> Header Element It describes the content at the top of the page <body>. It may include a logo, navigational links or a search bar.
<nav> Navigation Section Element It encapsulates the page’s navigational links. It is often placed inside the <header> or <footer>.
<main> Main Element It represents the dominant content of the <body> of a document.
<footer> Footer Element It includes the page’s footer content at the bottom of the <body>.
<section> Section Element It defines elements in a document, such as chapters, headings, or any other area of the document with the same theme. Generally developers will use <section> to define a theme for the webpage and use <article> to write independent content for that theme.
<article> Article Element It holds content that makes sense on its own such as articles, blogs, and comments.
<aside> Aside Element It is used to mark additional information that can enhance another element but isn’t required in order to understand the main content. Usually, this information would be in a sidebar or a location where it doesn’t obstruct the main piece of content. An example of this would be an article that discusses how to take care of a dog and next to the article an ad would appear advertising a dog grooming product.
<figure> Figure Element It is used to encapsulate media such as an image, diagram or code snippet.
<figcaption> Figure Caption Element It is used to describe the media encapsulated within the <figure> element. Developers will normally use <figcaption> within the <figure> element to group the media and description. This way, if a developer decides to change the position of the media, the description will follow along with it.
<audio> Audio Element It is used to embed audio content into a document.
<video> Video Embed Element It embeds a media player which supports video playback into the document.
src type controls Source, Type, Controls Attributes They specify the source, type of audio / video and will display audio / video controls in the media player.
<embed> Embed External Content Element It embeds external content at the specified point in the document. It is a deprecated tag and other alternatives, such as <video>, <audio> and <img>, should be used in its place, but is being taught for legacy purposes.
<strong> Strong Element It highlights important, serious, or urgent text and browsers will normally render this highlighted text in bold by default.
<em> Emphasis Element It emphasizes text and browsers will usually italicize the emphasized text by default.
